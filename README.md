# Mean Plan

A client-side web application for managing recipes, meal planning, and grocery shopping. No backend required - all your data stays with you.

  - Home-cooked recipes
  - Dine out
  - Pizza/takeout
  - Custom entries
- Adjust servings and copy meals to other days

### üõí Grocery Planning
- Auto-generate shopping lists from your meal plans
- Select time period (next week, 2 weeks, or custom range)

## Data Storage

Your data is stored locally in your browser (LocalStorage) with optional OneDrive sync for backup and cross-device access. No server means complete privacy and control over your data.

## Getting Started

1. Clone this repository
2. Open the app in your web browser
3. Start adding recipes and planning meals!

## Tech Stack

- Client-side only (no backend required)
- LocalStorage for data persistence
- OneDrive integration for cloud sync (optional)
